🛠️ Especificaciones Técnicas del ADN Jordano
El contrato rastrea la densidad de los caldos y la eficiencia de la desnaturalización proteica:

Concentración de Jameed (uint256): Mide la densidad del caldo lácteo rehidratado. Un valor de 90 indica una saturación coloidal perfecta para el Mansaf, donde el yogur seco de cabra actúa como medio de cocción estable.

Tiempo de Cocción en Horas (uint256): Registra la duración de la exposición térmica. En el Zarb, este tiempo refleja la eficiencia del horno subterráneo para romper fibras complejas mediante calor radiante.

Usa Arroz Safranado (bool): Validador de la base de carbohidratos aromatizada, esencial para la estructura de platos como la Maqluba.

Gobernanza: Sistema de feedback integrado mediante likes y dislikes.
